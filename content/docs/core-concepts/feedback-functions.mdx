# Feedback Functions

Feedback is how you tell Trace what's good or bad about your AI system's behavior.

## What is Feedback?

Unlike traditional machine learning which uses numerical losses, Trace accepts **any kind of feedback**:

- ✅ Numerical rewards or losses
- ✅ Natural language descriptions
- ✅ Test pass/fail results
- ✅ Compiler errors
- ✅ User preferences
- ✅ Multi-modal feedback
