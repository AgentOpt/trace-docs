# Installation

Get started with Trace in just a few minutes.

## Requirements

- Python >= 3.10
- pip or conda

## Install via pip

The easiest way to install Trace:

```bash
pip install trace-opt
```

## Development Installation

If you want to contribute or work with the latest version:

```bash
git clone https://github.com/AgentOpt/Trace.git
cd Trace
pip install -e .
```

## Optional Dependencies

### AutoGen Backend

For backward compatibility with AutoGen:

```bash
pip install trace-opt[autogen]
```

### Test Dependencies

For running tests:

```bash
pip install trace-opt[test]
```
