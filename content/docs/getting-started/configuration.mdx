# Configuration

Trace2 supports multiple LLM backends through LiteLLM and AutoGen.

## LLM Backend Setup

### Default: LiteLLM

By default, Trace2 uses [LiteLLM](https://docs.litellm.ai/) which supports 100+ LLM providers.

#### OpenAI

```python
import os
os.environ["OPENAI_API_KEY"] = "sk-..."
os.environ["TRACE_LITELLM_MODEL"] = "gpt-4o"
```

#### Anthropic

```python
import os
os.environ["ANTHROPIC_API_KEY"] = "sk-ant-..."
os.environ["TRACE_LITELLM_MODEL"] = "claude-3-5-sonnet-20241022"
```

#### Other Providers

See [LiteLLM documentation](https://docs.litellm.ai/docs/providers) for other providers.
